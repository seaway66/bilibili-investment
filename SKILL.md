# B站投资类视频投研报告 Skill

## 简介

专为 B 站**投资、财经、行业分析**类视频设计的深度分析工具。通过画面识别 + 语音转录完整还原 UP 主的研究框架、核心数据和投资逻辑，生成专业投研报告。

## 适用场景

- 股票/行业分析类视频 → 提取投资 Thesis 和标的分析
- 产业链/技术路线类视频 → 还原产业链结构和公司卡位
- 财务数据分析类视频 → 读取图表数据，汇总对比
- 投资策略/方法论类视频 → 还原分析框架和决策逻辑

## 与 bilibili-summary 的区别

| | bilibili-summary | bilibili-investment |
|------|:---|:---|
| 视角 | 通用内容总结 | **投资研究分析** |
| 识图 Prompt | "描述内容" | "列出所有数字、公司名、图表趋势" |
| 报告长度 | 中等 | 长（含产业链/财务/风险） |
| 数据表格 | 可选 | **必须**（市场/财务/参数三张表） |
| 免责声明 | 无 | **必须**（投资风险警告） |

## 工作流程

```
B站投资视频(BV号)
  → 下载视频 + 音频
  → 每18秒提取一帧 + 相似度去重
  → GLM-4.6V-Flash(免费) 逐帧识别(投资视角Prompt)
     └─ 识别不清 → 自动回退 qwen-vl-max(付费)
  → Whisper tiny 转录音频
  → AI 综合数据 → 生成投研报告
```

## 视觉识别策略：GLM 优先，qwen 兜底

**默认**：所有帧用 GLM-4.6V-Flash（免费）

**回退条件**（满足任一换千问）：
1. GLM 返回 429 限流
2. 输出 < 50 字或明显错误
3. 数据与其他帧矛盾
4. 核心数据帧（财务图表/产业链图）数值缺失


## 输出报告

报告保存在 `{output_dir}/{视频标题}_投研报告.md`，包含：

1. **投资 Thesis** — UP 主核心投资观点
2. **核心逻辑链** — 现象→推论→结论→标的
3. **行业/技术分析** — 背景、壁垒、产业链
4. **数据汇总** — 市场数据表、财务数据表、关键参数表
5. **涉及标的分析** — 每家公司逐一分析（产业链位置、财务亮点、风险点、评级）
6. **UP 主研究方法论** — 信息来源、分析框架、判断强度
7. **风险与不确定性** — UP 主承认的 + AI 发现的
8. **后续关注点** — 催化剂、跟踪指标、验证信号
9. **免责声明** — 投资风险警告
